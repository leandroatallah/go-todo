# Command-Line Golang TODO List

This is an TODO list application written in Golang to run on the Command-Line.

## Clone Repository

`git clone git@github.com:leandroatallah/go-todo.git`

## Build

Enter the folder and run the build:

`cd go-todo`

`go build .`

## Commands

- `list`
- `list --all`
- `add "<content>"`
- `remove <id>`
- `complete <id>`
